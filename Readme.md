A **simple, clean Express.js REST API** built in **JavaScript** that manages sales records and generates a ranked leaderboard.

## Project Overview

This backend system allows you to:
- Submit sales records via REST API
- Automatically aggregate multiple records per sales agent
- Generate a ranked leaderboard based on total sales and deals
- Handle tie-breaking with consistent rules


##  Installation & Setup

### Prerequisites
- Node.js 14+ installed
- npm package manager

### Step 1: Install Dependencies

```bash
npm install
```

### Step 2: Start the Server

```bash
npm start
```

The server will start at `http://localhost:3000`

### Step 3: Test It!

Open your browser and go to:
```
http://localhost:3000
```

You should see the welcome message!
